## ゲームの動機

このゲームは、SergeiがRobonomics Academyでの記事（[最初](https://robonomics.academy/en/learn/escape-from-black-mirror/overview/)および[2番目](https://robonomics.academy/en/learn/school-2024-iot-vs-ai/overview/)）で話したブラックミラーからの脱出に関するアイデアから多くのインスピレーションを得ています。

想像してください。普通の安いロボット掃除機が、その所有者の生活を監視する深刻な能力を持っているとします。悪意のある人物がこのデータにアクセスを得るかどうかは重要ではありません。ビデオ録画のAI分析だけで、人の生活の非公開の側面についての結論を導くのに十分です。汚い秘密、気づかれない行動パターン、個人情報など、すべてがプロファイルの炉に入ります。あなたが無料のトークンを手に入れようとしている間に考えてみてください。

## ゲームの目標

ゲームでは、プレイヤーはモバイルロボットのデータからシードフレーズを復元することが求められます。ロボットは研究室内をランダムに移動し、ビデオを記録します。時々、ロボットは12個のシードフレーズの単語のうち1つを公開します。ロボットが停止した際、プレイヤーはビデオで数字の書かれた看板を見ることができ、その単語がシードフレーズ内での順序を示唆します。課題は、看板の数字とロボットの停止時に受け取った単語のセットを照合し、正しいシードフレーズを復元することです。

最初にシードフレーズを復元し、そのアドレスからすべてのXRTトークンを引き出したプレイヤーが勝者となります。

## ゲームのルール

*プレイヤーの登録*

- 参加するには、プレイヤーはRobonomicsパラチェーンで新しいアカウントを作成する必要があります。このアカウントはEdwards（ed25519）暗号化タイプで作成する必要があります。手順は[当社のウィキ](https://wiki.robonomics.network/docs/create-account-in-dapp/)で利用可能です。
- 次に、プレイヤーはRobonomicsサーバーのDiscordボットに移動し、新しいセッションの開始についてのメッセージを待ちます。
- セッションが開始されると、プレイヤーはRobonomicsパラチェーン形式の作成されたアカウントアドレスをDiscordボットに送信します。プレイヤーは60分間入力する時間があります。

*研究室の探索*

- 登録が完了すると、モバイルロボットが研究室を探索します。その後、ロボットはビデオ録画とシードフレーズの単語のリスト（正しい順序ではない）を送信します。
- プレイヤーは、Discordボットからロボットの結果を受け取るメッセージを待ち、リンクを介してアプリケーションに移動してシードフレーズの復元を開始します。

*シードフレーズの復元*

- データにアクセスするには、プレイヤーは登録されたRobonomicsアカウントを使用する必要があります。
- プレイヤーはビデオ録画とシードフレーズの単語セットにアクセスできます。プレイヤーの課題は、研究室に配置された看板の数字に基づいて単語の正しい順序を見つけることです。

<TextCard alignContent="left">

**例.** もし単語のセットが次のような場合：

`finding -> church -> relation -> dog -> sample`

そして、ビデオから再構築されたロボットのルートが次の場合：

`看板 #5 -> 看板 #3 -> 看板 #1 -> 看板 #2 -> 看板 #4`

その場合：

`finding — 看板 #5 -> findingは5番目の単語`

`church — 看板 #3 -> churchは3番目の単語`

`relation — 看板 #1 -> relationは1番目の単語`

`dog — 看板 #2 -> dogは2番目の単語`

`sample — 看板 #4 -> sampleは4番目の単語`

結果として、正しいシードフレーズ：`relation dog church sample finding`

</TextCard>


- シードフレーズを見つけた後、プレイヤーは任意のアプリケーションを使用してトークンへのアクセスを復元できます。
- たとえば、Polkadot.js拡張機能では、*既存のシードからアカウントをインポート*機能を使用してこれを行うことができます。
- すべての復元されたアカウントは、ほとんどのPolkadot / Kusamaウォレットでサポートされている**Schnorrkel（sr25519）**暗号化タイプを持っています。

*勝者の発表*

- プレイヤーがアカウントにアクセスしたら、XRTを引き出してセッションの勝者になる必要があります。他の誰かがそれを行うまで。
- たとえば、Polkadot.js拡張機能を使用して、これはアカウントページの[当社のPolkadot / Substrateポータル](https://polkadot.js.org/apps/?rpc=wss%3A%2F%2Fkusama.rpc.robonomics.network%2F#/accounts)で行うことができます。送信ボタンを押してトークンを引き出し、任意の場所に送金します。全残高引き出しを行うために*Transfer the full account balance, reap the sender*チェックを使用できます。
- セッションが完了し、トークンが引き出された後、ボットはDiscordチャンネルに勝者のアドレスを公開して通知します。

## イベント条件

このイベントは9日間続き、毎日朝と夜（合計18回）に各セッションで**42 XRT**に最初に到達する機会があります。

セッションの登録は、毎日**10:00と20:00 EEST**に開始され、60分間続きます。

イベントの開始を見逃さないように、当社のソーシャルネットワークをフォローしてください！
