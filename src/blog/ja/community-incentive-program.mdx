Robonomics Web Servicesのベータテストが開始されたことで、Robonomics Networkは成長ペースを上げています。Robonomicsコミュニティに報いるため、Robonomics DAOはコミュニティインセンティブプログラムのアイデアを考えました。

<TextCard icon="/icons/icon-forum.png" link="https://discourse.robonomics.network/c/robonomics-web-services/">

### Robonomics Web Servicesについてまだよくわからないですか？
フォーラムをご覧ください！RWSの機能、ユースケース、制限事項について、コアRobonomics開発者と議論する場所です。

</TextCard>

> Robonomicsコミュニティインセンティブプログラムの目標は、**開発者**、**アンバサダー**、**インフルエンサー**に彼らの貢献に対して**XRT**を提供することです。

したがって、コアRobonomics DAOはマーケティング部門代表の管理下に10,000 XRTを割り当てました。
これらの資金はRobonomicsマーケティングDAOの管理下にあります。開発者に報酬を提供し、アンバサダーやインフルエンサーとコミュニケーションを取る責任があります。

ロボティクス業界の中核となる開発者、エンジニア、研究者は、Robonomics Networkに基づく研究開発活動に対して報酬を受け取ります。

開発者のインセンティブプログラムについて詳しくは[こちら](https://robonomics.network/blog/rewards-for-researchers-developers-and-engineers/)をご覧ください。

Robonomics Networkの情報を世界中に広め、世界的なロボット経済の認知度を高めましょう。

アンバサダーやインフルエンサー向けのインセンティブプログラムについて詳しくは[こちら](https://robonomics.network/blog/rewards-for-ambassadors-and-influencers/)をご覧ください。
