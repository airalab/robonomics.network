## 基本機能に慣れる

開発者がRobonomics Networkのコンセプトを理解する最良の方法は、[playground](https://wiki.robonomics.network/docs/create-account-in-dapp/)セクションを訪れて試すことです。

[playground](https://wiki.robonomics.network/docs/create-account-in-dapp/)セクションでは、Robonomics制御下でサイバーフィジカルシステムを起動するプロセスを案内します。参加者は、Robonomics Networkに関する説明を読むために提供された段落のいずれかを選択できます。

第1波の参加者は、以下のタスクを完了した場合に**200ドル（ネイティブRobonomicsトークンであるXRTの形式で）で報酬を受け取ります**:

* [playground](https://wiki.robonomics.network/docs/create-account-in-dapp/)セクションのウィキでステップバイステップの手順を読む;
* シミュレーションまたは実際のIoTデバイスまたはロボティクスシステムを起動する;
* YouTubeにビデオをアップロードし、ビデオ内で以下を説明する必要があります:
    * コメントとスクリーンキャストを使用した起動プロセスの動作方法;
    * IPFSのヘルプを借りてCPSのデータログをブロックチェーンに保存する方法;
* ツイートを作成し、@Aira_robonomics Twitterアカウントをメンションし、$XRTと#robonomicsnetworkをタグ付けします。

ツイートでは、参加者はRobonomics Networkでロボットを起動した経験を共有する必要があります。

## グローバルな大気汚染レベルをRobonomics Networkに保存する

Robonomics Networkは、市民観測ネットワークによる汚染レベルの公開と、人々が炭素排出量を相殺するのを支援することを目指しています。そのため、コアDAO開発者は、[水](https://www.frontiersin.org/articles/10.3389/frobt.2020.00070/full)および大気汚染センサーネットワーク、スマートビルの相殺、グリーン証明書を提案しました。

オープンソースセンサーネットワークの採用を推進するためにRobonomics Web Servicesチームは、次のタスクを完了した最初の15人の参加者に**$300（ネイティブRobonomicsトークンであるXRTの形式で）を報酬として提供することに決定しました:**

* ウィキの「Cases」セクションで[ステップバイステップの手順](https://wiki.robonomics.network/docs/connect-sensor-to-robonomics/)を読む;
* 大気汚染センサーをRobonomicsに接続し、1週間以上維持する;
* YouTubeにビデオをアップロードし、ビデオ内で参加者が説明する必要があります:
   * 大気汚染センサーをRobonomicsネットワークに接続するプロセスはどのように見えるか+スクリーンキャスト;
   * IPFSのヘルプを借りてブロックチェーンに大気汚染レベルを保存する方法;
* ツイートを作成し、@Aira_robonomics Twitterアカウントをメンションし、$XRTと#robonomicsnetworkをタグ付けします。

ツイートでは、参加者はセンサーの地理的位置情報を共有し、それをRobonomicsネットワークに接続する体験を共有する必要があります。

## Robonomics Networkでの研究/プルーフ・オブ・コンセプト・プロジェクトの作成

Robonomics Web Servicesは、スマートファクトリー、ロボティクスシステム、およびIoTデバイスのデジタルツインのシミュレーションを作成するための優れた環境です。
**インスピレーションの源をお探しの場合は、Robonomicsの科学的リソースを訪れることをお勧めします:**

> Robonomics Networkで行われた[研究開発](https://wiki.robonomics.network/docs/r-and-d-based-on-robonomics-network/);

> Robonomics Networkチームの中核メンバーによって行われた[科学的研究](https://robonomics.network/community#science);

上記のケースに関する質問がある場合は、Robonomics開発者フォーラムを訪れてください。

**Robonomics DAO IoT＆ロボティクス開発者は、あなたのプロジェクトを評価し、研究をサポートします。**
**支払額（プロジェクトごとに最大300 XRT）は、プロジェクトの難易度とあなたの自律性に応じてDAOによって決定されます。**

## Robonomics Web Servicesベータテストへの参加

Robonomics Web Services（RWS）は、IoT＆ロボティクスインフラの新時代をもたらします。
これは、開発者にAmazon Web Servicesと類似した機能を提供しますが、中央集権化されたクラウドベースではなく、顧客にブロックチェーンへのアクセスを提供します。

RWSの[利点について詳しくはこちら](https://blog.aira.life/robonomics-web-services-and-rws-token-intro-d730ab50ad42?source=collection_home---4------2-----------------------).

## RWS Beta テスト

私たちは、産業4.0の開発者に、彼らのIoT＆ロボティクスユースケースのためのDLTインフラストラクチャを提供することに興奮しています。

IoTデバイスからRWSにテレメトリを送信するすべての人に報酬が支給されます。
毎月末にRWSに転送されたテレメトリの量に基づいて、アクティブなトップ100参加者に1000 XRTが分配されます。
