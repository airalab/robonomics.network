CrustとRobonomicsが協力し、ロボティクスやスマートデバイス向けの堅牢なデータストレージインフラを提供することになりました。このパートナーシップにより、Kusama上のCrustとRobonomicsのパラチェーン間でより深い統合が実現し、ユーザー体験を簡素化するためのクロスチェーン連携に関する新しいアプローチが生まれます。

私たちが共に目指している主な目的は2つあります。

<ol>
    <li>
        IoT（モノのインターネット）やロボティクスデバイスから得られるデータの堅牢な保存。現在、Robonomicsパラチェーンは、ロボティクスコントローラーと中央集権型ピニングサービスの両方でIPFSを使用し、データの永続性を確保しています。Crustの分散型ピニングサービスにアクセスできることで、ユーザーは多くの冗長な保存場所での暗号学的な保存証明を得ることができます。
    </li>
    <li>
        dAppsの分散型ホスティング。これはこれまで利用できなかったものであり、そのためRobonomicsのdAppsは従来のクラウドプロバイダーや自社ハードウェアでホスティングする必要がありました。Crustを利用することで、dAppsは分散型ホスティングが可能となり、サービスの可用性が保証されます。開発者が単一障害点のないサービスを構築し、世界中のユーザーに常に利用可能な状態を維持できることは、ロボティクスや物理世界での展開全般にとって非常に重要です。
    </li>
</ol>

RobonomicsチームはすでにCrust Networkの調査を開始しており、丁寧なドキュメント作成の姿勢に感銘を受けています。すでに [Boston Dynamics robotin Bay Area lab](https://spot.merklebot.com) を Robonomics パラチェーンのトランザクションを通じて操作し、データは Crust に記録されます。ぜひ [spot.merklebot.com](https://spot.merklebot.com) でご自身でお試しください。

![Demo web app](/blog/images/robonomics-and-crust-networks-strategic-partnership/demo.png)

次に、Crust Shadow と Robonomics パラチェーン間の XCM 統合に注目します。これにより、スマートデバイスから完全に分散化された方法でストレージ用のトランザクションを直接送信できるようになります。サービスが一方のパラチェーンから他方に提供される際のトランザクション手数料に関する興味深い課題があることも注目すべき点です。

私たちは、ユーザーが利用しているパラチェーン内でのみトランザクションの支払いを行うべきだと考えています。今回の場合、Robonomics パラチェーンのユーザーがロボティクスやスマートデバイスからのデータを Crust Network に保存したい場合、XRT でのみサービスの支払いを行えばよいのです。

Robonomics パラチェーンは、Crust Shadow へデータ保存の要求と Robonomics パラチェーンアカウントから直接 CSM トークンで支払うことができる特別なパレットの導入に取り組んでいます。ユーザーはストレージコストを自動的に補うために XRT を支払うことになります。

最終的には、パラチェーンが DEX 上でネイティブトークンを自動的に売却し、追加サービスを提供する他のパラチェーントークンの残高を補充するようになります。この完全に分散化され自律的なシナリオがすでに現実的であることは驚くべきことです！

ロボティクス、IoT、スマートデバイス分野における真に分散化されたソリューションの共同開発に向けて、エキサイティングな未来を楽しみにしています。

## Crust について

Crust Network は、多目的に設計されたストレージブロックチェーンであり、Web3 エコシステム全体に対して真に分散化されたファイルストレージとホスティング機能を提供し、プライバシーとデータ所有権を強化します。Crust Network は、dApp ホスティング、NFT メタデータストレージ、暗号化ファイルストレージを幅広いブロックチェーンやビジネスケースに対応して実現します。

## Robonomicsについて

Robonomicsは、IoTアプリケーション向けのオープンソースプラットフォームです。私たちは、ユーザーアプリケーション、IoTサービス、複雑なロボティクス間で技術的および経済的情報の交換をアトミックトランザクションの形で実現する新世代のインターネット技術（Web3）をサポートしています。

Robonomicsは、ロボティクス側とユーザーインターフェース側の両方でIoTアプリケーションを開発するための完全なツールセットを備えています。ユーザーとデバイス間の通信は、Web3の世界で最も成功している技術、すなわちIPFS、Ethereum、Polkadotを使用して行われます。これにより、開発者はスマートホームからインダストリー4.0まで、現代的で安全なアプリケーションを作成することができます。
