**Robonomics Web Services（RWS）**は、産業、消費者、および商業ソリューション向けのIPFS、Ethereum、およびPolkadot上の分散型インフラストラクチャサービスです。

## 最初のRWS IoTコアサービスに会いましょう
RWSは、お使いいただくときに気に入る最新のインフラストラクチャです。RWSを使用すると、デバイスを簡単に接続し、ユーザーアプリケーション、クラウドサービス、および他のデバイスと安全にやり取りできます。単一障害点なし、オープンソース、最も興味深いインターネット技術をターンキーで提供し、世界中のどこでも自由に使用できます。ロボティクスとIoTデバイスのための真にグローバルなネットワークです。

Robonomics Parachain上の最初のRWS IoTコアサービス：

* **RWS: PubSubメッセージブローカー。** IPFSプロトコルに基づくP2Pネットワークを介した無制限のメッセージングを使用します。
* **RWS: データブロックチェーン化サービス。** データをIPFSに配置し、ハッシュをパブリックブロックチェーンに保存します。
* **RWS: ロボットアクション。** 分散コンピュータの制御下でIoTデバイスをオン/オフに切り替えたり、起動したり停止したりします。

## RWSトークンでサブスクリプションモデルの実験を開始しましょう
Robonomicsネットワークの主要トークンはXRTです。これは、ロボティクス/IoTデバイスが今日Ethereum上でやり取りするためのトランザクション手数料を支払うために使用できるユーティリティトークンです。近いうちに、XRTはRobonomics Parachain上でPolkadotエコシステムで同じ機会をロボティクスシステムに提供します。

エンジニア向けに、コア開発チームは異なるRobonomics使用モデルを試み、その最初のステップとしてライフタイムサブスクリプショントークンであるRWSトークンを立ち上げます。Teslaがロードスターをライフタイム充電でプロモートしたように、私たちは最初の方にRobonomicsネットワークをライフタイムサブスクリプションオファーでプロモートしたいと考えています。ロボット経済の先駆者になろう！

> ### 1 RWS トークンは、Robonomics Parachainを介してIoTデバイスから/への1秒あたりのトランザクション（tps）を提供します。
>
> RWS トークンをアクティブ化すると、1か月に2,500,000回以上の取引を時間制限なしで行うことができます。RWS トークンを非アクティブ化し、いつでもUniswapで取引することができます。

**ケース1の例:** あなたは0.1 RWS トークンを所有し、10秒ごとにPublic blockchainでシャドウとテレメトリを共有したいと考えています。これを実現するには、イーサリアムメインネットのスマートコントラクトの特別なアドレスにトランザクションを送信してアクティブ化する必要があります。その後、RWS: データブロックチェーン化サービスを10秒ごとに使用できます（1 RWS トークン == 1 tps、0.1 RWS トークン == 1 tx / 10秒）。

**ケース2の例:** あなたは2 RWS トークンを所有し、Robonomicsの制御下で自動車洗車を行いたいと考えています。これを実現するには、イーサリアムメインネットのスマートコントラクトの特別なアドレスにトランザクションを送信してアクティブ化する必要があります。その後、RWS: データブロックチェーン化サービスとRWS: ロボットアクションを並行して毎秒使用できます。

**RWS トークンの現在の状況:** 初期配布段階。

**総供給量:** 100 RWS トークン、これは100 tpsのRobonomics Parachain帯域幅の終身利用を表します。

**配布計画:**
- 50 RWS トークンは[Uniswap XRT/RWS ペア](https://uniswap.info/pair/0x42018a2ff89b6d662b5f746277bd02f0d85f485a)を通じて配布されます。
- 50 RWS トークンは学術機関および非営利団体向けに予約されます。

**RWS トークン機能の開始予定時期:**
- アルファ版 — 2020年第3四半期
- ベータ版 — 2020年第4四半期

**RWSの追加発行。** Polkadot Parachainsのベンチマークによると、100 tpsはRobonomics Parachainの潜在的な帯域幅の一部に過ぎません。同時に、コア開発チームはさまざまな種類のサブスクリプションを提供したいと考えています。私たちの視点からは、RWS トークンは先駆者向けの特別なオファーです。そして将来的には、ロボノミクスウェブサービスをロボティクスおよびIoTデバイスで使用する際の月次およびその他の種類の制限に焦点を当てる予定です。

## 結論
Robonomics Web Servicesは新しい分野です。Robonomicsプロジェクトの進化についての最新情報です。以前は、ソフトウェアプラットフォーム、分散ネットワークプロバイダー、および当社の技術に基づくいくつかのユースケースを持っていると述べていました。Robonomicsプロジェクトは、新しいインターネット技術の波の最先端に少なくともAmazon Web Servicesと同等以上のものをもたらすための一歩を踏み出しました。まだ研究開発段階で実験的ですが、新しい分野で最初になりたい先駆者は歓迎されます！RWS IoT Coreでのさらなるサービスやコア以外のサービスも、今年の夏に発表される予定です。アルファ版はできるだけ早く開始されます。
