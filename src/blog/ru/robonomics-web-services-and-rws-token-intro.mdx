**Robonomics Web Services (RWS)** - это децентрализованные инфраструктурные службы на основе IPFS, Ethereum и Polkadot для робототехники и интернета вещей. Для промышленных, потребительских и коммерческих решений.

## Знакомьтесь с первыми службами RWS IoT Core
RWS - это современная инфраструктура, которая вам понравится. RWS позволяет легко подключать устройства и взаимодействовать с пользовательскими приложениями, облачными службами и другими устройствами безопасно. Нет единой точки отказа, открытый исходный код, самые интересные технологии интернета "под ключ" и ваша свобода использовать их в любом месте мира. Действительно глобальная сеть для устройств робототехники и интернета вещей.

Первые службы RWS IoT Core на параблокчейне Robonomics:

* **RWS: Брокер сообщений PubSub.** Используйте неограниченную передачу сообщений по сети P2P на основе протокола IPFS.
* **RWS: Сервис блокчейнизации данных.** Помещайте данные в IPFS и храните хеши в публичном блокчейне.
* **RWS: Действия робота.** Включение / выключение; запуск или остановка ваших устройств IoT под управлением децентрализованного компьютера.

## Начнем эксперимент с моделью подписки на токен RWS
Основной токен сети Robonomics - XRT. Это утилитарный токен, который ваши устройства робототехники / интернета вещей могут использовать для оплаты комиссий за транзакции для взаимодействия через Ethereum сегодня. Вскоре XRT предоставит вашим системам робототехники такую же возможность в экосистеме Polkadot с помощью параблокчейна Robonomics.

Для инженеров команды Core Dev попробует создать различные модели использования Robonomics, и первым шагом в этом направлении станет запуск токена с пожизненной подпиской - токена RWS. Как Tesla продвигала свой родстер с пожизненной зарядкой, так и мы хотим продвигать сеть Robonomics с предложением пожизненной подписки для первых пользователей.Станьте пионером роботизированной экономики!

> ### 1 токен RWS даст вам 1 транзакцию в секунду (tps) от/к вашему устройству IoT через Robonomics Parachain.
>
> Просто активируйте свой токен RWS и совершите более 2 500 000 транзакций в месяц без ограничений по времени. Деактивируйте токен RWS и торгуйте им на Uniswap в любое удобное время.

**Пример случая 1:** У вас есть 0,1 токен RWS и вы хотите делиться телеметрией с тенью в публичном блокчейне каждые 10 секунд. Вам нужно активировать его с помощью транзакции на специальный адрес смарт-контракта основной сети Ethereum. После этого вы сможете использовать RWS: Сервис блокчейнизации данных каждые 10 секунд (1 токен RWS == 1 tps, 0,1 токен RWS == 1 транзакция / 10 сек).

**Пример случая 2:** У вас есть 2 токена RWS и вы хотите контролировать автоматическую мойку автомобилей под управлением Robonomics. Вам нужно активировать его с помощью транзакции на специальный адрес смарт-контракта основной сети Ethereum. После этого вы сможете использовать RWS: Сервис блокчейнизации данных и RWS: Действия робота каждую секунду в параллельном режиме.

**Текущий статус токена RWS:** начальный этап распределения.

**Общее количество:** 100 токенов RWS, которые представляют собой пожизненное использование пропускной способности Robonomics Parachain в 100 tps.

**Планы по распределению:**
* 50 токенов RWS будут распределены через [пару Uniswap XRT/RWS](https://uniswap.info/pair/0x42018a2ff89b6d662b5f746277bd02f0d85f485a).
* 50 токенов RWS будут зарезервированы для академических и некоммерческих организаций.

**Ожидаемое время запуска функций токена RWS:**
* альфа-версия — III квартал 2020 года
* бета-версия — IV квартал 2020 года

**Дальнейшая эмиссия токенов RWS.** Оценки Парабол Полкадота показывают, что 100 tps является небольшой частью потенциальной пропускной способности Robonomics Parachain. В то же время команда Core Dev хочет создать различные типы подписок. С нашей точки зрения, токен RWS представляет собой особое предложение для пионеров. И в будущем мы сосредоточимся на других типах токенов RWS с ежемесячными и другими типами ограничений на использование веб-сервисов Robonomics для робототехники и устройств IoT.
