随着Robonomics Web Services beta测试的推出，Robonomics Network正加快增长步伐。为了奖励Robonomics社区，Robonomics DAO提出了社区激励计划的想法。

<TextCard icon="/icons/icon-forum.png" link="https://discourse.robonomics.network/c/robonomics-web-services/">

### 对Robonomics Web Services还不确定吗？
访问我们的论坛！这是一个与核心Robonomics开发人员讨论RWS功能、用例和限制的地方。

</TextCard>

> Robonomics社区激励计划的目标是为**开发人员**、**大使**和**影响者**提供**XRT**作为奖励。

因此，核心Robonomics DAO分配了10,000 XRT，由市场部门代表管理。
这些资金将由Robonomics营销DAO控制。负责奖励**开发人员**并与**大使**和**影响者**进行沟通。

作为机器人行业的核心推动力量，开发人员、工程师和研究人员将根据Robonomics Network的研究和开发活动获得奖励。

了解更多关于开发人员激励计划的信息[在这里](https://robonomics.network/blog/rewards-for-researchers-developers-and-engineers/)。

在全球范围内传播Robonomics Network的声音，增加对全球机器人经济的认识。

了解更多关于大使和影响者激励计划的信息[在这里](https://robonomics.network/blog/rewards-for-ambassadors-and-influencers/)。
