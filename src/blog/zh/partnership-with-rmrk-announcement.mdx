[Kanaria](https://kanaria.rmrk.app/) 是世界上最先进的 NFT 项目，建立在世界上最先进的 NFT 协议 [RMRK](https://rmrk.app/) 上。
Kanaria NFT 是一种可以拥有甚至装备其他 NFT 的鸟类，构建了一个物品经济体系，不仅可以跨越不同项目（自动跨项目兼容性），还可以跨越不同链，因为它是在 Kusama 上推出的 - 一个让链通过连接的“第0层”。Kanaria 也是 RMRK 即将推出的元宇宙 - [Kanaria: Skybreach](https://url.rmrk.app/meta) 的前身。

Robonomics 承诺支持将 RMRK 的 Pallet 集成到他们的运行时中，从而与世界上最先进的 NFT 逻辑进行本地集成（此添加仍需社区治理投票批准）。这种集成使得 RMRK 2.0 NFT 能够在 Robonomics 即将推出的平行链上本地存在，并对基于 RMRK 标准构建的 NFT 在物联网应用和复杂机器人中的利用方式进行扩展和探索产生影响。

作为最早参与和构建波卡生态系统的项目之一，Robonomics 提供了一个平台，用于构建供应链、支付系统、生产、数字护照和*机器人即服务*等用例的 Web3 机器人解决方案和应用程序。

另一方面，RMRK 将奖励参与 [Robonomics 众贷](https://robonomics.network/kusama-slot/) 的前 500 名众贷者。这些冠军将获得一个特殊的 Kanaria 兼容物品 - 一个可以装备到自己的 Kanaria 鸟上的子 NFT，并可能在 [RMRK 的元宇宙](https://url.rmrk.app/meta) 中带来优势。

您不必拥有 Kanaria 鸟即可获得此奖励物品，但您必须是前 500 名贡献者之一。

## 关于 Robonomics

Robonomics 是一个面向物联网应用的开源平台。他们支持一种实现用户应用程序、物联网服务和复杂机器人之间以原子交易形式交换技术和经济信息的新一代互联网技术（Web3）。

Robonomics 平台提供了与机器人经济网络合作的工具。Robonomics 允许智能城市和工业 4.0 区域的设计者在自主机器人服务之间建立信任，通过 dapp 直接为从自主工厂和城市传感器网络订购产品的用户提供访问。这反过来使 Robonomics 及其用户能够建立一个全球监控网络物理系统活动的去中心化系统。

## 关于 RMRK

[RMRK](https://rmrk.app/) 是一个领先开发 RMRK 协议的团队，用于使用 [Kusama](https://kusama.network/) 上的 NFT 创建高度可组合的“NFT 乐高”。

除其他功能外，RMRK 还为 NFT 带来了以下特点：

* 在链上对 NFT 进行表达
* 自定义燃烧消息
* 拥有 NFT 的 NFT
* 具有条件渲染的 NFT（如果这样，显示那样）
* 具有升级槽的 NFT
* 具有多资源的 NFT（即包含游戏代码和高分辨率封面艺术）
* 可作为 DAO 进行治理的 NFT

因此，它远远超出了当前 ERC721 / ERC1155 标准及其衍生品的功能。
