## 熟悉基本功能

开发人员了解Robonomics Network概念的最佳方式是访问[playground](https://wiki.robonomics.network/docs/create-account-in-dapp/)部分并尝试。

[playground](https://wiki.robonomics.network/docs/create-account-in-dapp/)部分指导人们如何在Robonomics控制下启动网络物理系统。参与者可以选择任何提供的段落来了解Robonomics Network。

第一批参与者将获得**200美元（以本地Robonomics代币XRT形式）的奖励，以完成以下任务**：

* 阅读维基[playground](https://wiki.robonomics.network/docs/create-account-in-dapp/)部分中的逐步说明；
* 在模拟或实际环境中启动任何类型的物联网设备或机器人系统；
* 在YouTube上上传视频，在视频中参与者必须解释：
    * 启动过程如何运作，包括评论+屏幕录制；
    * 如何借助IPFS将CPS的数据日志存储在区块链中；
* 发一条推文并提及@Aira_robonomics Twitter账户并标记$XRT和#robonomicsnetwork。

在推文中，参与者应分享他在Robonomics Network上启动机器人的经历。

## 将全球空气污染水平存储在Robonomics Network中

Robonomics Network旨在通过全球市民观测网络揭示污染水平，并帮助人们抵消碳足迹。因此，核心DAO开发人员提出了[水](https://www.frontiersin.org/articles/10.3389/frobt.2020.00070/full)和空气污染传感器网络、智能建筑抵消和绿色证书。

为推动基于开源传感器网络的采用。在Robonomics Web Services上，团队决定奖励前15名参与者**$300（以本地Robonomics代币XRT形式）完成以下任务：**

* 阅读维基“Cases”部分中的[逐步说明](https://wiki.robonomics.network/docs/connect-sensor-to-robonomics/)；
* 将空气污染传感器连接到Robonomics并维护超过1周；
* 在YouTube上上传视频，参与者必须解释：
   * 将空气污染传感器连接到Robonomics网络的过程是什么样的+屏幕录像；
   * 如何借助IPFS将空气污染级别存储在区块链中；
* 发一条推文并提及@Aira_robonomics Twitter账户并标记$XRT和#robonomicsnetwork。

在推文中，参与者应分享其传感器的地理位置，并分享连接到Robonomics网络的经验。

Robonomics Web Services是创建智能工厂模拟、机器人系统和物联网设备数字孪生的绝佳环境。
**如果您正在寻找灵感来源，我们建议您访问Robonomics科学资源：**

> 在Robonomics Network上进行的[研究与开发](https://wiki.robonomics.network/docs/r-and-d-based-on-robonomics-network/)；

> 由核心Robonomics Network团队成员进行的[科学研究](https://robonomics.network/community#science)；

如果您对提到的案例有任何疑问，请访问Robonomics开发者论坛。

**核心Robonomics DAO物联网和机器人开发人员将评估您的项目并协助您进行研究。**
**DAO将根据项目的难度和您的自主权决定支付金额（每个项目最多300 XRT）。**

## 参与Robonomics Web Services测试版

Robonomics Web Services（RWS）将带来物联网和机器人基础设施的新时代。
它将为开发人员提供类似于亚马逊Web Services功能，但与集中式云基础不同，它将为客户提供访问区块链的功能。
