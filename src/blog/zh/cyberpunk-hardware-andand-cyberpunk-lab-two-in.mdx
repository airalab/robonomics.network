# 赛博朋克硬件 && 赛博朋克实验室 - Robonomics 为 Polkadot 社区推出的两项倡议   

当我们转向生产实验室时，我不想失去未来的研究潜力。回顾 2024 年夏天，很难想象如何实现这一目标。我专注于将团队内的所有研发成果转化为具有统一愿景的实际项目。这导致了赛博朋克硬件品牌的创建，仅在其推出后我们才开始着手研究机器人技术。事情变得更加复杂，但经过三个月，我们的团队现在已经接近将我们在机器人技术研发中积累的经验应用到生产公司的加密友好投资模型中。我们已经适应，今天，我想介绍两个计划，这些计划将有助于发展开源 OG Robonomics 项目及其基础设施。   

## 赛博朋克实验室 - Polkadot 人形机器人 SDK   

赛博朋克实验室是一个旨在推动人形机器人、人工智能和去中心化技术边界的大胆倡议。在美国加利福尼亚州旧金山和中国深圳设有确定的地点，这个研究实验室将作为 Polkadot 人形 SDK 的发射台，实现人形机器人与 Web3 生态系统的无缝集成。   

通过将尖端机器人技术与 Polkadot 云能力相结合，我们正在构建一个安全、自主和去中心化机器交互的框架。这个实验室将作为一个实践场所，研究人员、艺术家和开发人员可以在其中创建、测试和验证现实世界的应用程序，为下一个技术演进的阶段。   

### 我们与人形机器人的经验：   

- 在研究机器人领域超过10年。   
- 作为加密行业中唯一展示 Web3 应用在机器人领域的团队，自2015年以来。   
- 与以太坊、IPFS 和 Polkadot 开发了多个标准机器人软件的集成。   
- 2024年12月，我们收购了第一台人形机器人，并开始研发其功能。   

这一经验的最佳成果是为需要标准化云服务进行遥测传输、身份管理、加密支付甚至智能合约执行的开发人员开发了 SDK 工具集。我们将从多个基于 Polkadot 的应用程序中合并功能，确保开发人员可以快速将 Polkadot 集成到他们的人形机器人项目中，并且最重要的是，它将正常工作。   

为了实现这一目标，我们与加利福尼亚的几个项目合作，Vitaly 很快将详细介绍。我们将在旧金山和深圳设立两个实验室，以完善 Polkadot 人形机器人 SDK 的功能，并消除进入的最大障碍——“这个东西启动不了！”。我们的目标是确保 SDK 在流行的人形机器人模型中的可靠性。   

我们还与艺术家合作，并正在与大学进行合作研究，验证我们正在开发的 SDK 的质量。这种战略性的三管齐下的方法由那些知道如何完成任务的人领导。   

为了资助 Polkadot 人形机器人 SDK 的开发，我们已经与合作伙伴达成协议，并将通过 Polkadot 的链上治理请求额外资金支持。支持我们的提案！这对整个生态系统都是有价值的，而且成本比某些一次性事件要低。有了这笔资金，我们可以保留我们的研发团队，并将其努力引向一个雄心勃勃的倡议，对 Robonomics 和 Polkadot 生态系统都有益。   

## 赛博朋克硬件：具有 DAO 结构和投资者预测市场经济游戏的消费者科技品牌   

很快，我将发布一份关于 Robocorporation 的文件，概述资金和生产管理的新模式。这一模式基于去中心化自治组织（DAO）、预测市场和物联网（IoT）的原则。   

我们的目标是创建一个自主系统，其中投资和运营决策是去中心化的，由集体预测和来自连接设备的数据指导。这种方法确保了与传统公司相比更有效的资本分配。   

到4月初，我们将邀请加密社区加入 Roboc公司并希望获得资金来发展硬件品牌。我们的目标是推出一种能够持续制造设备和技术的结构，这些设备和技术基于我们团队和未来赛博朋克社区的开源蓝图和模型。

开源硬件是技术市场的未来，这就是为什么RISC-V已经开始受到关注。我们已经发布了我们的第一批设备样品，并正在积极吸引主要的DAO代币持有者。然而，我们的目标是建立一个海盗式公司——因此我们呼吁加密爱好者成为代币持有者。

敬请关注！一旦我们与主要投资者达成最终协议，并确认在未来两年内制造10-12台设备的支持，我们将正式邀请加入赛博朋克硬件社区。

最终，我们将让Robonomics栩栩如生，部署数百个人形机器人，并允许它们通过Polkadot提供服务，同时培育一个能够塑造消费者科技下一个时代的技术进步的社区！
