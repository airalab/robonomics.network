**Os Serviços Web Robonomics (RWS)** são serviços de infraestrutura descentralizada sobre IPFS, Ethereum e Polkadot para Robótica e IoT. Para soluções industriais, de consumo e comerciais.

## Conheça os primeiros serviços principais do IoT Core RWS
O RWS é uma infraestrutura de última geração que você vai adorar. O RWS permite conectar dispositivos facilmente e interagir com aplicativos de usuário, serviços em nuvem e outros dispositivos de forma segura. Sem ponto único de falha, código aberto, tecnologia de internet mais interessante de forma pronta para uso e sua liberdade para utilizá-la em qualquer lugar do mundo. Uma rede verdadeiramente global para dispositivos de Robótica e IoT.

Os primeiros serviços principais do IoT Core RWS sobre a Robonomics Parachain:

* **RWS: Corretor de Mensagens PubSub.** Use mensagens ilimitadas em uma rede P2P baseada no protocolo IPFS.
* **RWS: Serviço de Blockchainização de Dados.** Coloque dados no IPFS e armazene hashes em uma Blockchain pública.
* **RWS: Ações de Robô.** Ligue / desligue; inicie ou pare seus dispositivos IoT sob controle de um computador descentralizado.

## Vamos começar o experimento do modelo de assinatura com o token RWS
O token principal da rede Robonomics é o XRT. Este é um token de utilidade que seus dispositivos de robótica / IoT podem usar para pagar taxas de transação para interagir sobre o Ethereum hoje. Em breve, o XRT dará a seus sistemas de robótica a mesma oportunidade no ecossistema Polkadot com a Robonomics Parachain.

Para os engenheiros, a equipe principal de desenvolvimento tentará criar diferentes modelos de uso do Robonomics e o primeiro passo nesse sentido é lançar o token de assinatura vitalícia — token RWS. Assim como a Tesla promoveu seu roadster com carregamento vitalício, queremos promover a rede Robonomics com uma oferta de assinatura vitalícia para os primeiros.Torne-se pioneiro na economia dos robôs!

> ### 1 token RWS lhe dará 1 transação por segundo (tps) de/para seu dispositivo IoT na Robonomics Parachain.
>
> Basta ativar seu token RWS e fazer mais de 2.500.000 tx em um mês sem limites de tempo. Desative o token RWS e negocie-o no Uniswap sempre que desejar.

**Exemplo caso 1:** Você possui 0,1 token RWS e deseja compartilhar telemetria com a sombra na blockchain pública a cada 10 segundos. Você precisa ativá-lo com uma transação para um endereço especial do contrato inteligente da mainnet do ethereum. Depois disso, você pode usar o RWS: Serviço de Blockchainização de Dados a cada 10 segundos (1 token RWS == 1 tps, 0,1 token RWS == 1 tx / 10 seg).

**Exemplo caso 2:** Você possui 2 tokens RWS e deseja controlar uma lavagem de carro automática sob o controle da Robonomics. Você precisa ativá-lo com uma transação para um endereço especial do contrato inteligente da mainnet do ethereum. Depois disso, você pode usar o RWS: Serviço de Blockchainização de Dados e RWS: Ações de Robô a cada segundo em modo paralelo.

Ao mesmo tempo, a equipe principal de desenvolvimento deseja criar diferentes tipos de assinaturas. Do nosso ponto de vista, o token RWS é uma oferta especial para os pioneiros. E no futuro, nos concentraremos em outros tipos de tokens RWS com limitações mensais e outros tipos de uso dos Serviços Web da Robonomics para Robótica e dispositivos IoT.
