[Το Kanaria](https://kanaria.rmrk.app/) είναι το πιο προηγμένο έργο NFT στον κόσμο που βασίζεται στο πιο προηγμένο πρωτόκολλο NFT: [RMRK](https://rmrk.app/).
Τα NFT του Kanaria είναι πουλιά που μπορούν να κατέχουν και ακόμα και να εξοπλίζουν άλλα NFT, δημιουργώντας μια οικονομία αντικειμένων που μπορεί να επεκταθεί όχι μόνο σε διαφορετικά έργα (αυτόματη διασυνοριακή συμβατότητα έργων) αλλά και σε διαφορετικές αλυσίδες λόγω της εκκίνησής τους στο Kusama - ένα "Layer 0" για αλυσίδες που συνδέονται μεταξύ τους. Το Kanaria είναι επίσης ο πρόδρομος του επερχόμενου μετακόσμου της RMRK - [Kanaria: Skybreach](https://url.rmrk.app/meta).

Το Robonomics έχει δεσμευτεί να υποστηρίξει την προσθήκη των pallets της RMRK στο runtime τους, ενσωματώνοντας έτσι φυσικά την πιο προηγμένη λογική NFT στον κόσμο (η προσθήκη υπόκειται ακόμα σε ψηφοφορία κοινοτικής διακυβέρνησης). Αυτή η ενσωμάτωση επιτρέπει στα NFT του RMRK 2.0 να υπάρχουν φυσικά στο επερχόμενο parachain του Robonomics, με επιπτώσεις στην επέκταση και την εξερεύνηση του πώς τα NFT που βασίζονται στο πρότυπο RMRK μπορούν να χρησιμοποιηθούν σε εφαρμογές IoT και πολύπλοκη ρομποτική.

Ως ένα από τα πρώτα έργα που ασχολούνται και χτίζουν εντός του οικοσυστήματος του Polkadot, το Robonomics προσφέρει μια πλατφόρμα για τη δημιουργία λύσεων κινητικότητας Web3 και εφαρμογών για περιπτώσεις χρήσης στην αλυσίδα εφοδιασμού, συστήματα πληρωμών, παραγωγή, ψηφιακά διαβατήρια και *ρομπότ-ως-υπηρεσία*.

Από την άλλη πλευρά, το RMRK θα ανταμείψει τους κορυφαίους 500 συμμετέχοντες στο crowdloan του Robonomics [που συμμετέχουν στο crowdloan του Robonomics](https://robonomics.network/kusama-slot/). Αυτοί οι πρωταθλητές θα λάβουν δωρεάν ένα ειδικό αντικείμενο συμβατό με το Kanaria - ένα παιδικό NFT που μπορεί να εξοπλιστεί στο πουλί Kanaria και μπορεί να προσφέρει πλεονεκτήματα στο [μετακόσμο της RMRK](https://url.rmrk.app/meta).

Δεν χρειάζεται να έχετε ένα πουλί Kanaria για να λάβετε αυτό το μπόνους αντικείμενο, αλλά πρέπει να είστε στους 500 κορυφαίους συνεισφέροντες.

## Σχετικά με το Robonomics

Το Robonomics είναι μια πλατφόρμα ανοιχτού κώδικα για εφαρμογές IoT. Υποστηρίζουν μια νέα γενιά τεχνολογιών internet (Web3) που υλοποιεί την ανταλλαγή τεχνικών και οικονομικών πληροφοριών σε μορφή ατομικών συναλλαγών μεταξύ εφαρμογών χρηστών, υπηρεσιών IoT και πολύπλοκων ρομπότ.

Η πλατφόρμα Robonomics παρέχει εργαλεία για την εργασία με το δίκτυο οικονομίας ρομπότ. Το Robonomics επιτρέπει στους σχεδιαστές έξυπνων πόλεων και ζωνών βιομηχανίας 4.0 να χτίσουν εμπιστοσύνη μεταξύ των υπηρεσιών αυτόνομων ρομπότ, να παρέχουν άμεση πρόσβαση του χρήστη μέσω dapp για την παραγγελία προϊόντων από αυτόνομες εργοστάσια και υπηρεσίες αστικών αισθητήρων. Αυτό επιτρέπει στο Robonomics και στους χρήστες του να θεσπίσουν ένα αποκεντρωμένο σύστημα που παρακολουθεί παγκοσμίως τις δραστηριότητες των κυβερνο-φυσικών συστημάτων.

## Σχετικά με την RMRK

Η [RMRK](https://rmrk.app/) είναι μια ομάδα που ηγείται στην ανάπτυξη του πρωτοκόλλου RMRK για τη δημιουργία υψηλά συνθέσιμων "NFT legos" χρησιμοποιώντας NFT στο [Kusama](https://kusama.network/).

Μεταξύ άλλων, η RMRK φέρνει τα ακόλουθα χαρακτηριστικά στον κόσμο των NFT:

* αντιδράσεις σε NFT στην αλυσίδα
* προσαρμοσμένα μηνύματα καύσης
* NFT που κατέχουν NFT
* NFT που έχουν συνθήκες απεικόνισης (αν αυτό, εμφάνισε εκείνο)
* NFT που έχουν θέσεις για αναβαθμίσεις
* NFT που έχουν πολλαπλούς πόρους (δηλαδή περιέχουν τόσο κώδικα για ένα παιχνίδι όσο και υψηλής ανάλυσης εξώφυλλο τέχνης)
* NFT που μπορούν να διοικούνται ως DAOs

Έτσι υπερβαίνει κατά πολύ αυτό που μπορούν να κάνουν τα τρέχοντα πρότυπα ERC721 / ERC1155 και οι παράγωγές τους.
