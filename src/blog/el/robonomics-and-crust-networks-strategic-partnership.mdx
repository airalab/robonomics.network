Χαιρόμαστε να ανακοινώσουμε ότι η Crust και η Robonomics άρχισαν να συνεργάζονται για να φέρουν ανθεκτική υποδομή αποθήκευσης δεδομένων σε ρομποτικές και έξυπνες συσκευές. Αυτή η συνεργασία σημαίνει ότι θα υπάρξει πολύ βαθύτερη ενσωμάτωση μεταξύ των Crust και Robonomics parachains στο Kusama, καθώς και μια ενδιαφέρουσα νέα προσέγγιση για τη διασυνδετική συνεργασία για την απλούστευση της εμπειρίας χρήστη.

Υπάρχουν δύο κύριοι στόχοι που θέλουμε να επιτύχουμε μαζί:

Η ομάδα της Robonomics έχει ήδη αρχίσει να εξερευνά το δίκτυο Crust και έχει εντυπωσιαστεί από την συστηματική προσέγγιση στην τεκμηρίωση. Ήδη μπορείτε να εκκινήσετε ένα ρομπότ Boston Dynamics στο εργαστήριο Bay Area μέσω μιας συναλλαγής Robonomics parachain, ενώ τα δεδομένα θα καταγραφούν στη Crust. Μη διστάσετε να το δοκιμάσετε μόνοι σας στο [spot.merklebot.com](https://spot.merklebot.com).

![Demo web app](/blog/images/robonomics-and-crust-networks-strategic-partnership/demo.png)

Τώρα, θα επικεντρωθούμε στην ενσωμάτωση XCM μεταξύ των Crust Shadow και Robonomics parachains. Αυτό θα μας επιτρέψει να στέλνουμε συναλλαγές για αποθήκευση απευθείας από έξυπνες συσκευές με πλήρως αποκεντρωμένο τρόπο. Να σημειωθεί ότι υπάρχει ένα ενδιαφέρον σύνολο ερωτήσεων που σχετίζονται με τα τέλη συναλλαγής όταν οι υπηρεσίες παρέχονται από μια parachain σε μια άλλη, τα οποία πρέπει να απαντηθούν.

Πιστεύουμε ότι ο χρήστης θα πρέπει να πληρώνει μόνο για μια συναλλαγή εντός της parachain που χρησιμοποιούσε. Στην περίπτωσή μας, όταν ένας χρήστης της Robonomics parachain θέλει να αποθηκεύσει τα δεδομένα από ρομποτικές και έξυπνες συσκευές στο δίκτυο Crust - θα πρέπει να πληρώσει μόνο για τις υπηρεσίες σε XRT.

Η Robonomics parachain εργάζεται για την ανάπτυξη ενός ειδικού pallet που θα μπορεί να στείλει μια συναλλαγή στο Crust Shadow με την απαίτηση να αποθηκεύσει τα δεδομένα και να πληρώσει από τον λογαριασμό της Robonomics parachain απευθείας σε CSM tokens. Ο χρήστης θα χρεώνεται με κάποια XRT για να αποζημιωθεί αυτόματα για το κόστος αποθήκευσης.

Τελικά, οι parachains θα πωλούν αυτόματα τα native tokens σε ένα DEX για να αναπληρώσουν το υπόλοιπο των άλλων tokens parachain που προσφέρουν επιπλέον υπηρεσίες. Είναι εκπληκτικό να συνειδητοποιεί κανείς ότι αυτό το πλήρως αποκεντρωμένο και αυτόνομο σενάριο είναι ήδη ρεαλιστικό!

Ανυπομονούμε για ένα συναρπαστικό μέλλον συνεργαζόμενοι σε πραγματικά αποκεντρωμένες λύσεις στον τομέα της ρομποτικής, του IoT και των έξυπνων συσκευών.

## Σχετικά με την Crust

Το Crust Network είναι ένα ευέλικτο, ειδικά σχεδιασμένο blockchain αποθήκευσης που παρέχει πραγματικά αποκεντρωμένη αποθήκευση αρχείων και δυνατότητες φιλοξενίας για ολόκληρο το οικοσύστημα του Web3, ενισχύοντας την ιδιωτικότητα και την κυριότητα των δεδομένων. Το Crust Network επιτρέπει τη φιλοξενία dApp, την αποθήκευση μεταδεδομένων NFT και την κρυπτογραφημένη αποθήκευση αρχείων σε μια ευρεία γκάμα blockchains καθώς και επιχειρηματικών περιπτώσεων.
