Η ομάδα Robonomics θέλει να επηρεάσει την ανάπτυξη του δικτύου Kusama με σενάρια για προσωπική χρήση στο σπίτι σας! Θα δείξουμε: πώς μπορείτε να αλληλεπιδράτε με τους πιο δημοφιλείς αισθητήρες (όπως Aqara/Xiaomi) μέσω του Kusama, πώς μπορείτε να συνδέσετε ένα χώρο συνεργασίας ώστε να ελέγχεται από ένα Kusama Parachain, και πολλά άλλα…

Σύνδεσμος προς τον πίνακα προγράμματος του Decoded: https://decoded.polkadot.network/agenda/?date=2022-06-29&location=berlin&presentation=kusama-use-cases-for-your-home&tz=berlin

## Το πρόγραμμα του εργαστηρίου

### Μέρος 1: Εισαγωγή στο IoT 2022

Στην αρχή του μαθήματος, θα εξετάσουμε τα πιο δημοφιλή σενάρια όπου χρησιμοποιούμε το Διαδίκτυο των Πραγμάτων το 2022. Θα ρίξουμε επίσης μια ματιά στο πρότυπο σχήμα αλληλεπίδρασης μεταξύ εφαρμογών IoT στο τηλέφωνό σας και έξυπνων συσκευών στο σπίτι σας. Θα μάθουμε για τα τρέχοντα προβλήματα στην κατασκευή λύσεων IoT.

### Μέρος 2: Φώτα ανάβουν!

Ας συζητήσουμε την πιο δημοφιλή λύση ανοικτού κώδικα για το σπίτι και να περάσουμε από τα κύρια στάδια της εγκατάστασης, της ρύθμισης και να εξηγήσουμε πώς να συνδέσετε αισθητήρες στον βοηθό.

Στη συνέχεια, θα δοκιμάσουμε μια απλή αποκεντρωμένη εφαρμογή σε συνδυασμό με την επέκταση polkadot.js για να ανάψουμε ένα έξυπνο λαμπτήρα. Έπειτα θα στείλουμε την πρώτη συναλλαγή στο δίκτυο Kusama και θα μάθουμε πώς λειτουργούν οι συνδρομές IoT στο Parachain της Robonomics.

### Μέρος 3: Περίπτωση διαχειριστή συνεργασίας

Ας εξετάσουμε ένα πιο πολύπλοκο σενάριο από τον κόσμο του IoT. Είστε ο διαχειριστής ενός χώρου συνεργασίας στο Βερολίνο, αλλά φύγατε για 3 μήνες για να απολαύσετε διακοπές στη Βαλί. Πώς μπορούμε να οργανώσουμε ένα ασφαλές σενάριο για τη διαχείριση των χρηστών του χώρου συνεργασίας, όπου μπορούμε να λαμβάνουμε απομακρυσμένα και τακτικά δεδομένα από έξυπνες συσκευές, χωρίς να χρησιμοποιούμε σύνδεση με κεντρικές υπηρεσίες cloud.

### Μέρος 4: Περιπτώσεις χρήσης Robonomics σήμερα και αύριο

Στο τελευταίο μέρος του μαθήματος, θα μιλήσουμε για σενάρια εφαρμογών Web3 στον τομέα του IoT που έχουν ήδη ξεκινήσει. Μπορείτε να εξοικειωθείτε με αυτά τα σενάρια και να δοκιμάσετε να αλληλεπιδράσετε με αυτά μόνοι σας μετά το πέρας του μαθήματος.

## Προσκαλούμε όλους όσους ενδιαφέρονται να μάθουν για τη χρήση των τεχνολογιών Web3!

Για να συμμετάσχετε πλήρως στο μάθημα, θα χρειαστείτε ένα φορητό υπολογιστή με την [επέκταση polkadot.js](https://polkadot.js.org/extension/) εγκατεστημένη. Ωστόσο, προσκαλούμε επίσης ακροατές που θα ήθελαν να μάθουν περισσότερα για τη χρήση του Polkadot σε σενάρια από τον κόσμο του IoT και των ρομποτικών.
