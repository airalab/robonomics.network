化学の品質管理システムとしてのブロックチェーン

ethereumsmart servicedata storagequality control

元々 the following task was set developing a quality control system for the production of a certain chemical product

なぜここでqualityのmonitoringがそんなに重要なのですか

この化学製品の主な有効成分は二酸化塩素です 高濃度では健康に有害です そして濃度が通常より低い場合この化学製品は役に立ちません

ブロックチェーンはそれと何の関係があるの

Blockchainは製造会社への信頼構築に役立ちます The consumer knows that no one can change the information in the Blockchain That means that the manufacturing company can not forge the results of the audit

自律型サイバーフィジカルシステム（CPS）

この問題を解決するために各バッチからランダムに選んだパケットの濃度をチェックする自律型サイバーフィジカルシステム（CPS）が開発されましたそして検証には分光器が使用されます監査はテスト中は手動で実施されましたが生産ラインの登場によりマニピュレータを使って自動化することができます

CPSがどのようにRobonomicsのネットワークに組み込まれているかは興味深い

The quality control system is an autonomous economic agent that lists one single service on the market — quality control 品質管理システムは市場に唯一のサービスである品質管理を提供する自律的な経済主体です

私たちはCPS作業の次のシナリオを考えました

  1. 需要はサイト上で作成されます
  2. システムは対応する供給オファーを作成します
  3. AIRAは責任契約を作成します
  4. システムはタスクを受け取りそれを実行し始める
  5. 責任の確定
  6. 顧客は監査結果へのリンクが付いたQRコードを受け取ります

主な作業工程

The main work starts happening after receiving the message 'task' In our case the lab technician sees that a liability contract was created He then takes the next packet checks the concentration of chlorine dioxide and sends the file to the system When the file is received it's published in the IPFS network The hash of the file and the address of the liability contract are stored in the local database for quick access to information 主要な作業はメッセージ'task'を受信した後に始まります この場合ラボ技術者は責任契約が作成されたことを確認します その後次のパケットを取り二酸化塩素の濃度を確認しファイルをシステムに送信します ファイルが受信されるとIPFSネットワークに公開されます ファイルのハッシュと責任契約のアドレスは情報への迅速なアクセスのためローカルデータベースに保存されます

Finally the last message about the finalization of the liability is sent and tokens are transferred to the beneficiary If the lab technician is removed from the script the system becomes incorruptible The quality control system for the chemical product was designed and launched as part of this project For the record the architecture of the project turned out to be universal All it takes to change the algorithm of work after receiving the message 'task' is to implement the exact system at another production ついに責任の最終化に関する最後のメッセージが送信されトークンが受益者に転送されます ラボ技術者がスクリプトから削除されるとシステムは不正ができなくなります 化学製品の品質管理システムはこのプロジェクトの一環として設計され導入されました ちなみにプロジェクトのアーキテクチャは汎用的であることが判明しました 'task'というメッセージを受け取った後に作業アルゴリズムを変更するには別の生産現場で同じシステムを導入するだけで十分です

実装の詳細については次を参照してください 化学品質管理システムに関するブログ投稿.

品質管理のためのユニバーサルアーキテクチャ

化学製品の品質管理システムはこのプロジェクトの一環として設計され導入されましたプロジェクトのアーキテクチャは汎用的であることが判明しましたメッセージtaskを受け取った後に作業のアルゴリズムを変更するために必要なのは同じシステムを別の生産現場に導入することだけです
