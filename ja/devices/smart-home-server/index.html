オープンで効率的なハードウェア

私たちのホームサーバーは、RISC-Vプロセッサーを搭載し、完全に透明な回路基板の回路図を特長としています。RISC-Vは、BSDライセンス付きのオープンでモジュラーなプロセッサーアーキテクチャであり、プロプライエタリなARMとは異なり、ライセンス制限なしにカスタマイズの柔軟性を提供し、信頼性のないIoTデバイスやサーバーに最適です。

  • プロセッサ: StarFive JH-7110 with RISC-V 4-core CPU、2 MB L2 キャッシュ、最大 1.5 GHz、RV64GC ISA サポート
  • GPU: IMG BXE-4-32 MC1、最大600 MHz
  • メモリ: 8 GB LPDDR4 SDRAM、最大2800 Mbps
  • ストレージ: 32 GB TFカード（最小）
  • インターフェース: 2x Gigabit Ethernet ports, 4x USB 3.0 Type-A ports, 40-pin GPIO header, M.2 connector, eMMC socket, 2-pin fan header
  • パワー: USB-C経由で5V DC、最大30W（最小3A）または40ピンGPIO経由で5V入力
  • OS: Ubuntu 24.04 LTS

ベンダーに中立なスマートホームの中核

サーバーには、オープンソースのホームオートメーションプラットフォームであるHome Assistantがプリインストールされており、第三者のクラウドサービスに頼らずにスマートホームを完全に制御できます。

そのモジュラーデザインにより、さまざまなメーカーの3,000以上のデバイスをサポートし、ベンダーロックインなしでコンポーネント、自動化、および統合を簡単に追加できます。ZigbeeとWi-Fiプロトコルをサポートしているため、温度センサーからエネルギー消費モニターまで、さまざまなデバイスをローカルサーバーに接続できます。

プライバシーに配慮した設計

私たちのスマートホームサーバーは、企業の干渉を排除し、データをローカルに保持するか、個人の鍵で暗号化して分散型Web3ストレージに保存します。追跡や監視は一切なく、完全なプライバシーをお楽しみいただけます。スマートホームを安全かつ信頼できる方法でコントロールしてください。

Robonomics Web3クラウドとP2Pリモートコントロールを使用すると、スマートホームのすべての機能に暗号アクセスできます：

  • リモートで現在のデバイスの状態にアクセスします
  • パラチェーンまたはLibp2pを介してデバイスをリモートで制御します
  • ホームアシスタントのユーザーを管理します
  • Home Assistantの設定をバックアップして復元します
  • 自宅のデジタルツインを安全な分散クラウドに保存します
