Blockchain als Qualitätskontrollsystem für Chemie

ethereumsmart servicedata storagequality control

Ursprünglich war folgende Aufgabe gestellt worden die Entwicklung eines Qualitätskontrollsystems für die Herstellung eines bestimmten chemischen Produkts

Warum ist die Überwachung der Qualität hier so wichtig

Die Hauptwirkstoff dieses chemischen Produkts ist Chlordioxid Es ist in hohen Konzentrationen gesundheitsschädlich Und wenn die Konzentration unter dem Normalwert liegt dann ist dieses chemische Produkt nutzlos

Was hat Blockchain damit zu tun

Blockchain hilft Vertrauen für das Fertigungsunternehmen aufzubauen Der Verbraucher weiß dass niemand die Informationen in der Blockchain ändern kann Das bedeutet dass das Fertigungsunternehmen die Ergebnisse der Prüfung nicht fälschen kann

Ein autonomes Cyber-Physical System (CPS)

Ein autonomes Cyber-Physical System (CPS) das die Konzentration eines zufälligen Pakets aus jeder Charge überprüft wurde entwickelt um dieses Problem zu lösen Und ein Spektrograph wird zur Verifizierung verwendet Die Prüfung wurde während der Tests manuell durchgeführt aber mit dem Aufkommen der Produktionslinie kann sie mit einem Manipulator automatisiert werden

Es ist interessant wie ein CPS in das Netzwerk von Robonomics eingebaut wird

Das qualitätskontrollsystem ist ein autonomer wirtschaftsakteur der eine einzige dienstleistung auf dem markt anbietet — qualitätskontrolle

Wir haben das folgende Szenario der CPS Arbeit in Betracht gezogen

  1. Nachfrage wird auf der Seite erzeugt
  2. Das System erstellt ein entsprechendes Lieferangebot
  3. AIRA erstellt einen Haftungsvertrag
  4. Das System erhält die Aufgabe und beginnt sie umzusetzen
  5. Abschluss der Haftung
  6. Kunde erhält einen QR Code mit dem Link zu den Ergebnissen des Audits

Der Hauptarbeitsprozess

Die Hauptarbeit beginnt nach dem Erhalt der Nachricht 'task' In unserem Fall sieht der Labortechniker dass ein Haftungsvertrag erstellt wurde Er nimmt dann das nächste Paket prüft die Konzentration von Chlordioxid und sendet die Datei an das System Wenn die Datei empfangen wird wird sie im IPFS Netzwerk veröffentlicht Der Hash der Datei und die Adresse des Haftungsvertrags werden in der lokalen Datenbank gespeichert um schnellen Zugriff auf Informationen zu ermöglichen

Schließlich wird die letzte Nachricht über die Finalisierung der Haftung gesendet und Token werden an den Begünstigten übertragen Wenn der Labortechniker aus dem Skript entfernt wird wird das System unbestechlich Das Qualitätskontrollsystem für das chemische Produkt wurde im Rahmen dieses Projekts entworfen und eingeführt Zur Information stellte sich die Architektur des Projekts als universell heraus Alles was nötig ist um den Arbeitsalgorithmus nach Erhalt der Nachricht task zu ändern ist die genaue Implementierung des Systems in einer anderen Produktion

Für weitere Details zur Implementierung siehe die Blogbeitrag über das Chemie Qualitätskontrollsystem.

Universelle Architektur für Qualitätskontrolle
