자율 Cyber-Physical System (CPS)

이 문제를 해결하기 위해 각 배치에서 임의의 패킷의 농도를 확인하는 자율 사이버 물리 시스템(CPS)이 개발되었습니다 그리고 검증을 위해 분광기가 사용됩니다 테스트 중에는 감사를 수동으로 수행했지만 생산 라인이 등장함에 따라 매니퓰레이터를 사용하여 자동화할 수 있습니다

CPS가 Robonomics의 네트워크에 어떻게 구축되고 있는지 흥미롭다

The quality control system is an autonomous economic agent that lists one single service on the market — quality control

우리는 CPS 작업의 다음 시나리오를 고려했습니다

  1. 수요는 사이트에서 생성됩니다
  2. 시스템은 해당 공급 제안을 생성합니다
  3. AIRA는 책임 계약을 생성합니다
  4. 시스템이 작업을 받고 이를 실행하기 시작합니다
  5. 책임의 확정
  6. 고객은 감사 결과에 대한 링크가 포함된 QR 코드를 받습니다

The main work starts happening after receiving the message 'task' In our case the lab technician sees that a liability contract was created He then takes the next packet checks the concentration of chlorine dioxide and sends the file to the system When the file is received it's published in the IPFS network The hash of the file and the address of the liability contract are stored in the local database for quick access to information 주요 작업은 'task' 메시지를 받은 후에 시작됩니다 우리의 경우 실험실 기술자는 책임 계약이 생성된 것을 확인합니다 그 후 다음 샘플을 가져와 이산화염소 농도를 확인하고 파일을 시스템에 전송합니다 파일이 수신되면 IPFS 네트워크에 게시됩니다 파일의 해시와 책임 계약의 주소는 정보에 빠르게 접근할 수 있도록 로컬 데이터베이스에 저장됩니다

Finally the last message about the finalization of the liability is sent and tokens are transferred to the beneficiary If the lab technician is removed from the script the system becomes incorruptible The quality control system for the chemical product was designed and launched as part of this project For the record the architecture of the project turned out to be universal All it takes to change the algorithm of work after receiving the message 'task' is to implement the exact system at another production 마침내 책임의 최종화에 대한 마지막 메시지가 전송되고 토큰이 수혜자에게 이전됩니다 실험실 기술자가 스크립트에서 제거되면 시스템은 부패할 수 없게 됩니다 화학 제품의 품질 관리 시스템은 이 프로젝트의 일환으로 설계되고 출시되었습니다 참고로 프로젝트의 아키텍처는 보편적으로 판명되었습니다 'task' 메시지를 받은 후 작업 알고리즘을 변경하는 데 필요한 것은 다른 생산 현장에 동일한 시스템을 구현하는 것뿐입니다
