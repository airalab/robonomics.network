生产质量控制平台“Feecc”

polkadot ecosystemdata storagesmart servicequality control

将近4年了有一个长期的实验正在进行以将Robonomics技术应用于现实生产业务 FEECC是一个去中心化的制造数据积累系统能够实现关键生产阶段的可追溯性并确保数据的完整性这些数据由于基于区块链的不可篡改性而无法被更改或替换

从咖啡实验到生产质量控制

FEECC背后的想法始于一次咖啡实验 如果卖家能为你的卡布奇诺提供不可辩驳的真实性证明会怎样 如果这不仅仅是一杯咖啡而是汽车制造商的大型供应链呢 或者更有责任感的是一家医疗设备公司

Web3 magic IPFS 和区块链存储

录制软件是通过一个大红按钮和一个二维码标签打印机触发的但在表面之下有一整套Web3魔法首先视频并没有存储在常规的文件系统中而是使用了IPFS其背后的关键优势是所谓的CID内容标识符一个固定长度的密钥可以在全世界范围内找到你的文件理论上如此

第二且最重要的是CID被放入了Datalog——这是Robonomics Parachain的一个pallet功能，它允许你在区块链上永久存储任何最多512字节的字符串！这意味着通过将与内容相关的指针放入加密支持的数据存储中，你确保了对那杯咖啡的透明度和信任。你可能并不需要每次喝你最喜欢的lavanda raf时都用到它，但在某一天它尝起来很苦时，你和卖家都可以完全确定这里的罪魁祸首是谁

不仅是客户服务还是一个内部MES模块

MES 代表 Manufacturing Execution System 它负责制造过程的优化和分析并且有一个特定的地方 FEECC 可能会派上用场 看这里有一家医疗设备公司 EndoStars 使用手工组装的部件这些部件有出现缺陷的风险 在安装 FEECC 之前你根本无法判断缺陷是供应商的责任还是装配工的失误 不仅如此还有一些流程需要优化比如以数字形式提供装配方案而不是在大量纸张中查找

组装过程

装配工获取他们的零件并通过一个网页应用按钮触发录制过程（这个应用也包含装配说明）然后他们获得一个唯一的条形码贴在零件上并在视频监控下进行装配当装配过程完成时就像咖啡一样他们获得存储在IPFS中的数据以便分发并将CID放入Robonomics链中永久保存可选地他们可以为客户获取一个二维码（与数据库无关）但大多数用例是为内部质量控制部门

质量控制部门

在将货物发往客户之前他们会进行质量检查这个过程也通过一些FEECC web模块进行并且协议也会被保存在链上但这里的关键点是产品与其装配数据（目前为视频）的链接这个链接是不可替代的因为数据在区块链上而地址也与内容本身相关联有了这个质量控制部门可以确保同事的清白装配工人也可以保护自己免受指责

建筑

FEECC让你以不可替代的形式保存有价值的数据以避免欺诈并确保安全的供应链这可能还包括其他具有相同核心理念的制造模块即安全透明的数据存储和使用

这些数字是什么

The coffee experiment started 4 years ago and the medical manufacture integration occurred in fall 2021 With all that we can definitely say that FEECC is a real-life-proved Web3 application to provide secure data accumulation and storage for productions 咖啡实验始于4年前 医疗制造集成发生在2021年秋季 综合以上我们可以肯定地说 FEECC是一个经过现实验证的Web3应用程序 用于为生产提供安全的数据积累和存储

现实生活验证的Web3应用

FEECC 是一个基于 Robonomics 构建的用于创建数字产品护照和制造质量控制的系统 它实现了关键生产阶段的可追溯性 允许在大规模制造中精确检测错误 并通过基于区块链的不可篡改性确保数据的完整性无法被更改或替代
