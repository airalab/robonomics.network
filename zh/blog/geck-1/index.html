项目 G.E.C.K.：实施动机和首份研发报告

April 16, 2024 in 研发 / All posts

Robonomics 网络架构师 Sergei Lonshakov [EnsRationis] 的特别报告

在过去的两年里，我注意到关于网络状态和数字游牧主题的出版物和各种现象增加了。去年，我对“网络国家阿姆斯特丹”会议表现出极大兴趣。在阿姆斯特丹反思了几天后，我启动了一个名为 G.E.C.K. 的项目。

六个月后，我们的团队在巴厘岛的丛林中进行了家庭自动化的现场试验。今天，我想分享我们关于创建一个由 NFT 访问令牌所有者控制的自主家庭形象工作当前状态的第一份报告。我们的概念是，这样一个智能家居形象将允许加密社区将现实世界物体整合到 Web3 世界中，以创建各种形式的网络状态。

“G.E.C.K. - 是来自《辐射》世界的东西吗？”

对于出生在上个千年的人来说，G.E.C.K. 这个词可能听起来非常熟悉。许多读者，包括我自己，在《辐射 2》的荒野中度过了数百个小时。搜索那个本应该拯救阿罗约故乡的宝贵手提箱。对于不熟悉《辐射2》情节的人，让我简单解释一下G.E.C.K.是什么：

伊甸园创世工具包（G.E.C.K.）是一个手提箱，里面装有在荒野条件下建立文明所需的一切。根据《辐射》的创作者，每个避难所都应该配备这样一个G.E.C.K.手提箱。有了这个工具，幸存核冬天的人们可以比现实世界允许的更快地恢复文明。

“让我们去掉关于核冬天的部分，想象一下新一代人从庇护所中走出来，创造新形式的合作和睦邻。这是一个复杂的挑战，在地球的任何地方都很少见。除了历史上在欧洲自由城市中看到的自由斗争、香港特殊地位周围的事件，以及哥本哈根的社会实验——克里斯蒂安尼亚，很少有社会能持续超过一个世代。我为什么相信现在有机会呢？”

Web3正在改变公共机构领域的游戏规则

我对形成小型社区、网络结构以及在地球上任何地方尝试创建主权国家的历史进行了个人分析，发现最大的挑战在于为新社区提供基本的社会权利和责任，以及建立关键的公共机构。这并不仅仅是关于拥有一个能够制定大量次级法规的强大政府，而是确保民事关系法、财产所有权登记以及保证各方履行义务的保存。这些是至关重要的功能，应该融入到任何人类社区中，无论多么微小。当然，这适用于你思考这个社区将在20或40年内如何生存的情况。

在私人对话中，我总是不厌其烦地强调比特币的重要性，它不仅仅是电子现金，更是一个整个社会制度，确保了这个金融系统的功能性。同样，我认为以太坊是第一个可以执行合同并保证执行条款的合法平台。除此之外，还有提供基于创建独立所有权登记的服务的平台，DeFi世界中更复杂的金融服务，预测市场，保险解决方案，在 Polkadot 中进行链上治理实验，所有这些加在一起，看起来像是一个用于启动众多网络状态实验的平台的出现。 然而，在这些解决方案中，我尚未看到的一件事，也是一个必要的约束实体，就是在周围世界的物理对象与将存在于 Web3 云中的社会制度之间建立连接的实现。

Robonomics 充当网络状态和地球上物理位置之间的连接纽带

我现在不想深入探讨形成网络状态的所有方面，比如首先需要哪些具体的智能合约模板，哪种数字货币值得拥有主要地位，以及许多其他方面。我的兴趣在于我们团队拥有最大专业知识的领域，即将物理对象连接到 Web3 世界。自2015年以太坊推出以来，我一直专注于研究各种机器人、智能设备，用于实现智能家居控制场景、自动售货机或使用智能合约或专门设计的 Web3 环境进行无人机交付的场景的连接方案和实验。

这确实是 G.E.C.K. 项目今天的主要目标 - 开发使用一组智能设备来支持位于巴厘岛丛林或加勒比海岸边的小型自治村庄的日常生活的场景…或者您的想象力允许的任何地方，甚至是火星！

我们已经开发并运营了一个Web3云，用于在Polkadot生态系统中连接智能设备超过2年。在此期间，我们已编写了所有必要的代码，以连接受其数字对应物控制的物理智能家居。

我们当前的任务是组装一组实际的设备，我们认为这些设备对于安装在任何由数字游牧民获得所有权NFT代币的集装箱住宅中至关重要。

以下是我们的第一个这样的设备组，经过在巴厘岛丛林中一位老密码朋克朋友的家中工作后组装而成。

2024年3月在巴厘岛的实验

巴厘岛丛林中的一座火山斜坡。计划在未来几年在这里建造一个自治村庄。2024年3月，一组工程师前往这里，通过登巴萨的交通骑摩托车来协助创建未来Cybervalley的第一个自治家庭。该设施目前正在建设中，计划于2025年启动。

G.E.C.K.版本0.1

在我们在巴厘岛的工作之后，我们为密码朋克自治家庭的第一个版本智能设备套件开发了一个初始概念。

以下这是我们计划组装成一个套件（手提箱）的设备详细清单，将在实验室环境中尽可能进行定制，并提供类似宜家风格的用户手册，供用户在自己的智能家居中进行自助设置。这些套件将从2024年夏季开始提供。

G.E.C.K.套件第一个版本包括以下内容：

服务设备

  • 用于本地智能家居服务器的单板计算机 x1
  • 用于将太阳能电池板和电池连接到智能家居系统的单板计算机 x1
  • 用于与Web3云同步而无需中介的迷你服务器 x1
  • IoT 网关（Zigbee2MQTT）x1
  • 室外Wi-Fi路由器 x2
  • 室外Wi-Fi桥接器 x1
  • 5G互联网路由器
  • 智能家居应用平板 x3
  • 电力线扩展器 x2

室内非侵入式设备

  • 室内空气质量监测套件：
    • 温湿度传感器 x3
    • CO2传感器 x3
    • PM颗粒传感器 x1
  • 烟雾探测器 x2
  • 燃气探测器 x1
  • 运动传感器 x5
  • 光传感器 x2
  • 门窗开合传感器 x2
  • 噪音传感器 x2
  • 振动传感器 x1
  • 漏水传感器 x2
  • 存在探测器（基于无线电波，需常量220V电源）x2
  • 智能按钮 x3
  • 智能灯泡套装 x5
  • 通用红外控制器 x3

室内侵入式设备

-智能门锁 x1

  • IP摄像头 x2
  • 能源监测设备 x3
  • 智能水阀门 x2
  • 墙壁开关 x5
  • 智能插座 x5
  • 锅炉智能按钮 x1
  • 恒温器 x3

  • 水箱水位传感器 x2
  • 单独的恒温散热器阀门（如果计划使用供暖系统） x3

  • 自动室内植物浇水系统 x1

  • 水质传感器套装 x1

辅助消耗品和工具

在发货前准备，我们将：

  • 预先配置

  • 设置Home Assistant的模板安装，配置接口、房间和面板。

  • 准备消耗品以供立即使用。

  • 安装并启动设备。

  • 创建一个Robonomics账户，订阅并激活与本地智能家居服务器的集成。
  • 升级！
