Blockchain comme système de contrôle qualité pour la chimie

ethereumsmart servicedata storagequality control

À l'origine la tâche suivante a été fixée développer un système de contrôle de la qualité pour la production d'un certain produit chimique

Pourquoi la surveillance de la qualité est-elle si importante ici

La principale substance active de ce produit chimique est le dioxyde de chlore Il est dangereux pour la santé à des concentrations élevées Et si la concentration est inférieure à la normale alors ce produit chimique est inutile

Quel est le rapport avec la Blockchain

La blockchain aide à instaurer la confiance pour l'entreprise de fabrication Le consommateur sait que personne ne peut modifier les informations dans la blockchain Cela signifie que l'entreprise de fabrication ne peut pas falsifier les résultats de l'audit

Un système cyber-physique autonome (CPS)

Un système cyber-physique autonome (CPS) qui vérifie la concentration d’un paquet aléatoire de chaque lot a été développé afin de résoudre ce problème Et un spectrographe est utilisé pour la vérification L’audit a été effectué manuellement lors des tests mais avec l’apparition de la ligne de production il peut être automatisé à l’aide d’un manipulateur

C'est intéressant de voir comment un CPS s'intègre dans le réseau de Robonomics

Le système de contrôle qualité est un agent économique autonome qui propose un seul service sur le marché — le contrôle qualité

Nous avons envisagé le scénario suivant de travail CPS

  1. La demande est créée sur le site
  2. Le système crée une offre d'approvisionnement correspondante
  3. AIRA crée un contrat de responsabilité
  4. Le système reçoit la tâche et commence à la mettre en œuvre
  5. Finalisation de la responsabilité
  6. Le client reçoit un code QR avec le lien vers les résultats de l'audit

Le processus de travail principal

Le travail principal commence après la réception du message 'task' Dans notre cas le technicien de laboratoire voit qu'un contrat de responsabilité a été créé Il prend ensuite le paquet suivant vérifie la concentration de dioxyde de chlore et envoie le fichier au système Lorsque le fichier est reçu il est publié dans le réseau IPFS Le hash du fichier et l'adresse du contrat de responsabilité sont stockés dans la base de données locale pour un accès rapide à l'information

Enfin le dernier message concernant la finalisation de la responsabilité est envoyé et les jetons sont transférés au bénéficiaire Si le technicien de laboratoire est retiré du script le système devient incorruptible Le système de contrôle qualité pour le produit chimique a été conçu et lancé dans le cadre de ce projet Pour mémoire l’architecture du projet s’est avérée universelle Il suffit de mettre en œuvre le même système dans une autre production pour changer l’algorithme de travail après avoir reçu le message task

Pour plus de détails sur la mise en œuvre voir le article de blog sur le système de contrôle qualité en chimie.

Architecture universelle pour le contrôle de la qualité

Le système de contrôle qualité pour le produit chimique a été conçu et lancé dans le cadre de ce projet L’architecture du projet s’est avérée universelle Il suffit de mettre en œuvre le même système dans une autre production pour changer l’algorithme de travail après avoir reçu le message 'task'
