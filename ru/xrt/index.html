Практическое использование

XRT можно использовать для покупки IoT подписки которая гарантирует приоритетную обработку IoT транзакций устройства или пользователя Все транзакции в рамках IoT подписок имеют приоритет перед транзакциями где оплачивается разовый сбор

Таким образом для пользователей которые планируют регулярно использовать парачейн и для тех для кого важно иметь предсказуемые параметры задержки для своих IoT устройств близкие к одному периоду блока выгодно выбрать модель подписки

During the activation process of the IoT subscription the payment amount is determined by an ultra-fast auction lasting 10 blocks (~1-2 min) The auction model allows the parachain to autonomously offer the most fair price to the user for activating the subscription

Плата за активацию подписки IoT сжигается полностью по завершении текущего аукциона подписки IoT

Другие вопросы

Что такое RWS?

В процессе разработки основной функциональности Для инженеров команда Core Dev постарается создать различные модели использования Robonomics и первым шагом на этом пути станет запуск токена пожизненной подписки - токена RWS Каждый токен RWS даст вам 1 транзакцию в секунду (tps) от/к вашему IoT устройству через Robonomics Parachain Активируйте свой токен RWS и совершайте транзакции с "бесплатным газом" с предсказуемой задержкой перед выполнением Деактивируйте токен RWS и торгуйте им на DEX

Что такое AIRA/AIR ID?

Токен Pre-XRT распределен в 2017 как автономная часть дальнейшей инфраструктуры Robonomics на Ethereum После успешного запуска Robonomics на Ethereum в 2019 токены AIRA были подключены к созданной инфраструктуре проекта через смарт-контракт ambix который гарантировал автономный и независимый процесс обмена pre-XRT на ERC-20 XRT
