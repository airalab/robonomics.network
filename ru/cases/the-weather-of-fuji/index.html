Демонстрация "Получить погоду горы Фудзи"

ethereumsmart service

Отправьте запрос и получите результат от автономного агента. Результирующий файл сохраняется в IPFS, и для результирующего сообщения требуется подпись с использованием закрытого ключа.

Обзор

Этот кейс-стади исследует, как сеть Robonomics облегчает децентрализованный доступ к данным с использованием устройств IoT и технологии блокчейн. Путем использования децентрализованного приложения Robonomics, подключенного к метеостанции на горе Фудзи, пользователи могут получать данные о погоде в реальном времени безопасно и прозрачно.

Как это работает?

Требования:

  • Расширение MetaMask (Скачать здесь)

Шаги для получения данных о погоде:

  1. Откройте dApp: Посетите Датчик погоды Fuji страница в приложении Robonomics.

  2. Установите MetaMask (при необходимости): Если MetaMask не установлен, вам будет предложено установить его, чтобы dApp работал правильно.

  3. Разрешить подключение: Подключите свой кошелек MetaMask к приложению Robonomics по запросу.

  4. Запросить данные сенсора: Нажмите кнопку "Запросить текущие значения", чтобы инициировать запрос данных.

  5. Подпишите сообщение: Подтвердите запрос данных, подписав сообщение через MetaMask. Не требуется токены или эфир.

  6. Процесс сбора данных: Автономный агент соберет данные о погоде с датчика на горе Фудзи и передаст их обратно. Подождите ответа.

  7. Получить результат: dApp загрузит файл результата с IPFS (InterPlanetary File System), децентрализованной системы хранения.

  8. Просмотреть данные о погоде: Полученные метеоданные с горы Фудзи будут отображены в интерфейсе dApp.

Технические идеи

  • Децентрализованная коммуникация: Запрос данных о погоде обрабатывается с использованием протокола Robonomics, обеспечивающего прозрачное общение без посредников.
  • Хранилище IPFS: Результаты безопасно хранятся на IPFS с криптографической подписью, обеспечивающей целостность данных.
  • Автономные агенты: Умные контракты и автономные агенты облегчают безопасный сбор и передачу данных.

Ключевые преимущества

  • Доверительный поиск данных: Не требуется центрального органа для проверки данных.
  • Безопасность: Данные криптографически подписаны с использованием закрытого ключа агента.
  • Прозрачность: Все взаимодействия и результаты неизменно сохраняются в блокчейне и IPFS.
