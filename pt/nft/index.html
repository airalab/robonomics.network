    Estas coleções sazonais de NFT representam um ativo digital único que captura a interação exclusiva entre o usuário e o Spot dentro da parachain Robonomics.   4. 4. coleção de telescópios automatizados

    Cunhagem sob demanda

    coleção de telescópios automatizados

    Localizado no Deserto do Atacama, no Chile, um telescópio físico pode ser acessado por meio de um aplicativo descentralizado para capturar imagens em tempo real de objetos astronômicos no espaço profundo. Todo usuário com conexão à internet pode acessá-lo com tokens $STRGZN, que estão disponíveis para compra no aplicativo usando um cartão de crédito. Selecione um objeto astronômico, envie um comando para o telescópio tirar uma foto e aguarde o NFT cunhado na plataforma Singular. Por favor, observe que o aplicativo só está disponível para uso das 23h às 6h GMT-3 devido ao horário de funcionamento do telescópio.
